# 🎰 早餐抽卡系统

一个有趣的抽卡系统，帮你决定今天早餐吃什么！

## ✨ 特性

- 🎲 单抽和十连抽功能
- 🃏 精美的卡牌翻转动画
- ✨ SSR卡牌发光特效
- 📊 实时统计抽卡数据
- 📱 响应式设计，支持移动端

## 🎯 稀有度系统

- **N（普通）** - 50% 概率：豆浆油条、白粥配咸菜等
- **R（稀有）** - 30% 概率：小笼包、牛肉面等
- **SR（史诗）** - 15% 概率：广式早茶、日式拉面等
- **SSR（传说）** - 5% 概率：海鲜自助、米其林早餐等
